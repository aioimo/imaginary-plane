What is being graphed?

Z0 = 0

Zn+1 = Zn2 + c

The lines on the graph are a representation of the recursive sequence defined above, to the right, plotted on the complex plane.

In words: the first term is 0, and we calculate any other term by squaring the previous term, and adding the constant c.

c is whichever coordinate your mouse is hovering over.

For example

If we let c = 1, then the terms of our sequence would be:

Z0 = 0
Z1 = 02 + 1 = 1
Z2 = 12 + 1 = 2
Z3 = 22 + 1 = 5
Z4 = 52 + 1 = 26
...

And so on to infinity.

Or if we let c = i, then we have:

Z0 = 0
Z1 = i
Z2 = -1 + i
Z3 = -i
Z4 = -1 + i
Z5 = -i
Z6 = -1 + i
...

And so on, oscillating between the two values -i and -1 + i.

These terms form the vertices of the shape that is plotted when your mouse is hovering over a given c.

What is i?

What number, when multiplied with itself, gives -1? No real number has this property, since all real numbers,

Mathemeticians define the square root of -1 to be i.

In other words, we can say i2 = -1.
