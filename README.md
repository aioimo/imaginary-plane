# Imaginary Plane

Shows the iterated sequence on the complex plane generated by the rule:

_Z<sub>n+1</sub> = Z<sub>n</sub><sup>2</sup> + c_ <br>
_Z<sub>0</sub> = 0_

The sequence is called the "orbit" around _c_.
