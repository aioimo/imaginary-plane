# Imaginary Plane

The iterated sequence of complex numbers given by the rule:

_Z<sub>0</sub> = 0_  <br>
_Z<sub>n+1</sub> = Z<sub>n</sub><sup>2</sup> + c_ 


is called the "orbit" around _c_.
